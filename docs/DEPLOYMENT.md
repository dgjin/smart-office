# 智慧办公空间管理系统 (SmartOffice) - 部署说明书

## 1. 环境准备
在开始部署前，请确保您的开发环境满足以下条件：
*   **Node.js**: v18.0.0 或更高版本。
*   **npm/pnpm**: 建议使用 pnpm 以获得更快的依赖安装速度。
*   **Gemini API Key**: 您需要从 Google AI Studio 获取一个有效的 API Key。

---

## 2. 安装步骤

### 2.1 获取源码
将项目文件解压或从仓库克隆到本地目录。

### 2.2 安装依赖
在项目根目录下执行以下命令：
```bash
npm install
# 或者使用 pnpm
pnpm install
```

---

## 3. 环境变量配置
系统需要 API Key 才能驱动 AI 推荐和冲突智能处理功能。
1.  在根目录下创建一个 `.env` 文件（或直接在运行环境设置）。
2.  添加以下配置：
```env
API_KEY=您的_GEMINI_API_KEY
```

---

## 4. 运行与构建

### 4.1 开发环境运行
启动 Vite 开发服务器：
```bash
npm run dev
```
启动后，通常可以通过浏览器访问 `http://localhost:5173`。

### 4.2 生产环境构建
生成优化后的静态资源：
```bash
npm run build
```
构建产物将存放在 `dist` 目录下，您可以将其部署到 Nginx、Vercel 或任何静态托管平台。

---

## 5. 注意事项
*   **全屏显示**：本应用采用 `h-screen` 布局设计，建议在 PC 端 Chrome 浏览器中全屏预览以获得最佳效果。
*   **摄像头权限**：扫码签到功能需要摄像头权限。如果通过 `http` 访问，浏览器可能会禁用此权限，建议通过 `https` 部署。
*   **数据存储**：由于本系统默认使用 LocalStorage，清除浏览器缓存会导致本地设置的数据丢失。建议定期使用“数据中心”的备份功能导出 JSON。
