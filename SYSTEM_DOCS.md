
# 智慧办公空间管理系统 (SmartOffice) - 需求说明书及操作手册

## 1. 系统概述
智慧办公空间管理系统（SmartOffice）是一款面向现代化企业的物理资源管理平台。系统集成了会议室预订、工位申请、多级审批流程自定义、组织架构管理以及基于 Gemini AI 的智能选址建议。

---

## 3. 功能模块说明

### 3.1 账户、登录与个性化 - [更新]
- **多渠道登录**：用户可以使用注册的“企业邮箱”或“手机号码”进行登录。
- **强制改密**：系统强制要求初次登录或正在使用初始密码的用户必须修改密码。
- **多维主题方案**：侧边栏提供 5 种专业配色方案（商务蓝、翡翠绿、活力橙、胭脂红、极光紫），用户可随心切换，系统自动记忆选择。

### 3.2 组织架构智能导入 - [更新]
- **层级自动识别**：部门批量导入 CSV 现在支持“父子关系”自动关联。
- **导入格式**：`部门名称,上级部门名称(可选)`。
- **逻辑增强**：即便上级部门不在数据库中，只要它出现在当前导入列表的前序行中，系统也能正确构建树形结构。

### 3.3 数据全景看板 - [更新]
- **今日实时动态**：展示各资源在当天的预订分布，色块上直接显示 `姓名 · 部门`。
- **资源占用全景**：前瞻性展示未来 30 天内所有资源的预订密度。色块内已实现信息透出，无需悬停即可直观看到谁在什么时间占用了资源。

## 4. 操作手册

### 4.1 如何切换系统风格？
1. 在侧边栏的最下方找到“系统主题风格”区域。
2. 点击不同的彩色圆点。
3. 系统界面颜色将实时更新（包括按钮、进度条、图标高亮等）。

### 4.2 如何通过 CSV 导入有层级的部门？
1. 准备 CSV 数据，示例：
   ```csv
   部门名称,上级部门名称
   集团总部,
   研发中心,集团总部
   架构组,研发中心
   运维组,研发中心
   ```
2. 进入“部门管理” -> “批量导入”。
3. 粘贴上述文本并执行导入。系统将自动生成 三级 组织架构。
